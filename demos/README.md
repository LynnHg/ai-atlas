## Demo 目录规范

每个 demo 使用独立文件夹，包含：
- `README.md` — 项目说明、运行方式、依赖
- 源代码文件
- 依赖管理文件（如 `requirements.txt`、`package.json`）

命名格式：`主题/demo-名称/`，例如：
- `agent/multi-agent-demo/`
- `rag/simple-rag-demo/`
- `mcp/custom-mcp-server/`
